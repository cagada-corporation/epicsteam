Este proyecto fue creado utilizando **Markdown** y **Editor.md** para aprender a documentar repositorios de GitHub de manera profesional.

# ⚙️ Instalación

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/TU-USUARIO/epicsteam.git
```

---

## 2️⃣ Entrar a la carpeta

```bash
cd epicsteam
```

---

## 3️⃣ Crear entorno virtual

```bash
python -m venv venv
```

---

## 4️⃣ Activar entorno virtual

### Windows

```bash
venv\Scripts\activate


```

---

## 5️⃣ Instalar dependencias

```bash
pip install django pillow
```

---

## 6️⃣ Aplicar migraciones

```bash
python manage.py migrate
```

---

## 7️⃣ Ejecutar servidor

```bash
python manage.py runserver
```
